## 开发

- 安装依赖： `pnpm install`
- 运行项目： `pnpm tauri dev`
- 构建项目： `pnpm tauri build`

## 提示

如果您在构建过程中遇到网络错误，请在终端上启用代理

- `set http_proxy=http://127.0.0.1:7891`
- `set https_proxy=http://127.0.0.1:7891`
